# config
    * webpack 相关的配置文件夹(基本不需要修改)* index.js : 指定的后台服务的端口号和静态资源文件夹

    *依赖
# src
    * 主要写的东西都写在 src 下面

# static
    *   静态资源文件夹
# .babelrc ：的配置文件

# .eslintignore ：eslint 检查忽略的配置
# .eslintrc.js ：eslint 检查的配置
# .gitignore : git 版本管制忽略的配置

# package.json  应用包的配置文件
